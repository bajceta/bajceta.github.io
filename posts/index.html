Installing Void Linux on IBM Power 740

Going for Void Linux Recently I switched from Ubuntu to Void Linux on all my boxes. Fortunately Void has great support for PPC. I even have it installed on a Powerbook G3. For running Linux on PPC you need to know if your CPU support little endian or big endian. PowerPC started as BE and only with Power8 did it fully support LE mode. For a Power7 we need Void Linux ppc64 that is a BE version.
