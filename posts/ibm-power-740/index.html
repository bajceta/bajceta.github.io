March 27, 2022

I saw this on server on a auction with just a few minutes to spare. Always liked PPC for some reason, so it peaked my interest. Ad was sparse with info, 2 cpus, 384GB, no disks. No clue on which CPUs, was it really 384GB ram… Anyway I went for it.

Turns out it was a top of the line box.

Specs:

  • 2 x 8 core Power7 @3.55GHz
  • 384GB ram
  • 4 x 146GB 15k SAS

All CPUs and ram activated and has a PowerVM license. Power consumption is not so good, it idles at around 360w-400w. I’ll try pulling out some drives and PCIe cards that sit unused. It might take the consumption down by 20-40w.

IBM Power terminology

PowerVM - Hypervisor, like KVM or VMware on x86.

LPAR - Logical partition, what we would today call a VM.

Hardware Management Console - IBM appliance or VM that is used to manage virtualization of PowerVM

VIOS - Virtual IO Server, AIX instance that shares physical resources as virtual ones to other LPARs.

IBM Power7 servers always run a hypervisor, even when all resources are dedicated to one LPAR.

Uses for this server?

Since getting AIX is hard, it might serve as a build server for Void Linux PPC BE packages, since the maintaner can’t afford the time to continue. Not sure if I can, but I like the thought.
